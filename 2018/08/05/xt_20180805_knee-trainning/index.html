膝关节治疗过程跟踪
发布于 2018-08-05 / 更新于 2019-01-27 / Body Health / 热度 0 / 喜欢 0

历程

2018-04-22，彩霞医院，左膝关节正侧位X光

左膝关节相邻诸骨端骨性结构未见异常，关节间隙未见变窄，关节面光滑整齐。

2018-06-28，高新医院，左膝关节MRI平扫

左膝：（1）左膝外侧半月板前、后角损伤I°；（2）左膝后交叉韧带后方小囊肿；（3）左膝关节少许积液。

2018-07-01，红会医院，运动膝踝科

左膝髌下脂肪垫炎
处理：

  1. 适量运动
  2. 热敷
  3. 联苯乙酸、脉血康

2018-07-29，红会医院，运动膝踝科体格检查

左膝髌骨研磨试验阳性，左膝髌腱止点压痛阳性。左膝髌骨软化症、左膝髌腱止点末端病
处理：

  1. 休息、静蹲训练
  2. 氟比洛芬凝胶贴膏、塞来昔布胶囊

评估问题记录

  • 右侧脚踝蹲下时向下塌；
  • 左腿长，后面高；
  • 左侧髌骨向内；
  • 左腿有点o型，远离轴线；
  • 呼吸模式，利用腹压多点；
  • 锁骨高低不一致；
  • 腿的内旋不够20度；
  • 左边足功高，向外多一点；
  • 走路脚尖翘，跟腱和小腿后侧太紧，拉伸。

病情实验室

  1. 知乎子墨，半月板内侧后角变性、后交叉韧带肿胀，疼痛可能和十字韧带损伤有关，半月板已经开始损伤，需要调理

  2. 北京陈，髌骨损伤

训练 2018.07

  1. 放松
    • 髌骨放松，50次，一天2-3次；
    • 泡沫轴放松，大腿前侧，单腿。

  2. 拉伸

    • 脚掌撑墙，核心收紧，保持15s，3次；
    • 弯腰向前，屁股向后向上，肩向前，后侧肌肉；
    • 躺着，腿伸直45度，骨盆向下，踝向下收，外侧肌肉；
    • 坐下，腿伸开，背直，向下弯腰，内侧肌肉。

  3. 核心训练，基础腹式呼吸；

  4. 核心训练，进阶腹式呼吸，手掌向上，脚跟45度向上顶。

训练 2018.09.16

  1. 热身
    • 泡沫轴放松：大腿前侧后侧、小腿前侧后侧、臀肌、腰方肌（每个部位20-30次，慢速、痛点集中刺激）
  2. 核心训练
    • 腹式呼吸（10mins）
    • 平板支撑（3*1min）
  3. 臀部训练
    • 单腿臀桥（臀大肌）：14-16次，4-6组
    • 弹力带棒式开合、侧步走（臀中肌）：14-16次，4-6组
  4. 膝关节稳定性训练
    • 单腿下压长弹力带拉，做对抗：1分钟，4-6组，各个方向拉
  5. 股四头肌训练
    • 坐姿抬腿（负重），健身房有器械，外旋
    • 深蹲，外旋，起来时不要完全伸直
  6. 拉伸
    • 1的全部内容
    • 臀肌拉升：筋膜球滚、仰卧搭腿拉伸（15-30s）
    • 股四头肌的拉伸：站立拉伸or跪姿拉伸（15-30s）
